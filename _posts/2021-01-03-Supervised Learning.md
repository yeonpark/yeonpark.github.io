# First step into Machine Learning

We can categorize Machine Learning into three parts: 
1. Supervised Learning (SL)
2. Unsupervised Learning (UL)
3. Reinforcement Learning (RL)

![SML Logic Diagram](/img/SML Logic Diagram.png){: width="100%" height="100%"}
Origin: https://github.com/amueller/odscon-2015

The main difference between SL and UL is whether there is labelled data or not.

As we can see in the diagram, in the training section we make a model using the training data set and corresponding labels.

Next, we insert the test data set into the model for the prediction and finally evaluate test label values.

In the evaluation step, we check the accuracy of the model that we've used and we need to avoid the problem of overfitting and underfitting.

![Overfitting Underfitting Diagram](/img/Overfitting Underfitting Diagram.png){: width="100%" height="100%"}
Origin: https://tensorflow.blog/

Underfitting: Too simplified model is used therefore can't discover the features and diversity of the dataset.
Overfitting: Too complexed model is made and inappropriate to generalize. Maybe due to outliers.
